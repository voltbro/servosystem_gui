Программа тестировалась только на Ubuntu 24.04

## Установка: <br>
```
cd ~ <br>
git clone https://github.com/voltbro/servosystem_gui.git <br>
cd servosystem_gui <br>
./install.sh <br>
sudo reboot <br>
```
## Запуск <br>
```
cd servosystem_gui <br>
source .venv/bin/activate  <br>
python3 main.py <br>
```
